# Платформа онлайн-курсов - Колледж связи №54

Веб-приложение для управления образовательными курсами с системой регистрации студентов и административной панелью.

**Действующая версия:** https://aiusers.ubase-cloud.ru/

## О проекте

Платформа предоставляет полный цикл управления учебными курсами:
- Для студентов - просмотр курсов, запись на обучение, личный кабинет
- Для администраторов - полное управление курсами, статистика, аналитика

## Основной функционал

### Для студентов:
- Просмотр каталога курсов с фильтрацией
- Поиск курсов по названию и описанию
- Запись на интересующие курсы
- Личный кабинет с отслеживанием прогресса
- Управление своими подписками на курсы

### Для администраторов:
- Полное управление курсами (добавление, редактирование, удаление)
- Просмотр статистики платформы
- Управление форматами обучения (онлайн, очно, гибрид)
- Анализ популярности направлений

## Технологии

- Frontend: HTML5, CSS3, JavaScript
- Backend: PHP
- База данных: MySQL
- Сервер: Apache
- Стили: Modern CSS с Flexbox/Grid

## Для разработчиков

### Локальная установка (если требуется):

1. Клонируйте проект в папку веб-сервера
2. Настройте базу данных MySQL
3. Импортируйте SQL-структуру из раздела "База данных"
4. Настройте подключение к БД в config.php
